> **请勿输入真实敏感信息，纯属娱乐，谨防电信诈骗！**

## ✨ 核心亮点

- 💎 **高档视觉设计**：采用拟态毛玻璃（Glassmorphism）、星空背景漂浮光影动画以及动态交互，伪装成极度正规的 Fintech 巨头网站，与搞怪内容形成强烈反差。
- 📈 **搞笑指数滚动**：顶部的跑马灯循环显示“韭菜大盘指数”、“行长跑路概率（稳步上升）”等离谱指标。
- 📢 **大佬存款滚动播报**：LED大屏幕滚动显示虚构大佬的离奇存款记录。
- 💸 **智能吞钱终端**：输入卡号（带智能分段格式化和爆笑校验）与存入金额后，体验戏剧性的“瑞士地下金库连接”、“为行长购置避税岛别墅”等存钱步骤，并伴随满屏金币雨及纸钞飞舞特效。
- 🧾 **搞笑交易凭证**：存钱成功后生成精美的“交易超度凭证”，盖有“只存不取”的貔貅大印，并温馨提示“前方排队提现人数：14亿”。
- 💬 **维权互助聊天室**：本地模拟实时公共聊天室，保安大爷、保洁阿姨以及维权会长在里面实时吐槽，且支持智能识别敏感词（如“提现”、“利息”、“跑路”）并自动秒回神吐槽。

## 🚀 快速启动

本项目采用纯前端 HTML/CSS/JS 开发，零后端依赖，下载后直接双击即可运行。

### 方式一：直接双击运行
直接在浏览器中打开 `index.html` 即可。

### 方式二：使用 Python 启动本地服务器（推荐）
如果您需要确保 Lucide 图标和音频效果的最佳兼容性，可以使用命令行在项目目录下启动轻量级服务器：

```bash
python3 -m http-server 8000
```
然后在浏览器中访问 [http://localhost:8000](http://localhost:8000)。
